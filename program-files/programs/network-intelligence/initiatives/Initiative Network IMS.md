# Initiative Network IMS

## Purpose

Build the initial NIMS implementations.

Covers the following concerns:

1. Ansible infrastructure code
2. Uptime metrics for all network devices
3. Real-time traffic analysis
4. Event monitoring and alerting

## Objectives

### 2026 — [AUTO-82](https://utinternational.jira.com/browse/AUTO-82)

- **How may we reliably monitor the network traffic in our operation, using modern, automated systems, that can be integrated with other applications we use.** ([AUTO-88](https://utinternational.jira.com/browse/AUTO-88))

  This is our first production deployment of Prometheus and Grafana for monitoring, and integrate device inventory and configuration with Netbox. These will form the foundation of our network information management system moving forward.

### 2025 — [AUTO-54](https://utinternational.jira.com/browse/AUTO-54)

- Investigation in Netbox plugin development, IT and NIMS Architecture design, and initial development of Ansible infrastructure code. ([AUTO-69](https://utinternational.jira.com/browse/AUTO-69))
